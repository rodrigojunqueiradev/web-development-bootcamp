Diferences between Flexbox and Grid:

  • Flexbox:
    • A tool that helps me align content along a one dimensional line
  • Grid:
    • A tool for laying out content along a two dimensional grid

Grid + Flexbox or Flexbox + Grid

Knowing how to use different tool sets gives you the flexibility to create things exactly the way you want it to!

The more skills we have, the better we'll be able to adapt to situations and the easier things will be.
